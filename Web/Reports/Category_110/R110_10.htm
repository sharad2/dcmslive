This report displays detail of the pickslips for the given Customer or PO or Status. You can also see VAS related pickslip details through  this report. Same report can also be used for manual routing of EDI orders.Weight and Volume will only be displayed if boxes are already created for the orders.

Overview

Open order details can be seen by this report. For running this report at least one filter is must from the following filters :- Customer, PO and Reporting Status*. Report can be modelled as per the need by using following filter combinations:- Pickslip Import Date, Ship Date, Virtual Warehouse, VAS and Building. As the list of open orders can be too huge so this report is limited to show only 1000 rows. There is also a summary grid “Summary of the customers” available with this report where summary level information’s per customer can be seen. Comprehensive picture of Pickslips of all the PO# belonging to specific customer is presented by this report.

Note  :- Report will show only 1000 rows but while exporting data to excel all rows resulting from the applied filters will get exported.

The report is further helping in the following scenarios:-

  1. Detail of orders in different status can be seen by this report.
  2. VAS implementation and management can also be handled by using this report.
  3. User can also plan manual routing for EDI custoemrs through this report.

Data retrieval for EDI Manual Routing:-

This report will help on data retrieval which is required for manual routing. For this user will have to choose the option “Orders Required Routing” along with all other desired parameters. In this case this report will get switched to full data retrieval mode which means report will retrieve all the data because the limit of 1000 rows will not get applied when the option “Orders Required Routing” remain checked.

 

Algorithm

We are using the PO and DEM_PICKSLIP tables to fetch open pickslips and retrieving the box SKU details from tables BOX and BOXDET. We are using table SKUCASE to get the box case volume and empty weight.

*Reporting Status: The following reporting status are used in this report. The Definition of different reporting status is as follows:

  • In Order Bucket: Those pickslip for whom bucket is not created means processing of those pickslip is not started. i.e. status  is 1.
  • Under Process: Those pickslip which are in process and pickslip status is Pending or Hold or Null.
  • Validated: Those pickslip which boxes are passed from validation program and but not added in the BOL. i.e. picking status is completed and shipping id is null.
  • Ready To Ship: Those pickslip which is ready to ship i.e. picking status is completed and shipping id is not null.
  • Others: Those pickslip which are not belong in above define status. i.e. Other-picking _status.

Detail grid query. 
  WITH OPEN_PICKSLIP_DATA AS
 (SELECT PS.PICKSLIP_ID            AS PICKSLIP_ID,
         PS.PO_ID                  AS PO_ID,
         PS.CUSTOMER_ID            AS CUSTOMER_ID,
         PS.CUSTOMER_STORE_ID      AS CUSTOMER_STORE_ID,
         PS.CUSTOMER_DC_ID         AS CUSTOMER_DC_ID,
         PS.REPORTING_STATUS       AS REPORTING_STATUS,
         PS.PRINT_DATE             AS PRINT_DATE,
         PS.ONLOT_NUMBER           AS ONLOT_NUMBER,
         PS.VWH_ID                 AS VWH_ID,
         PO.START_DATE             AS START_DATE,
         PO.DC_CANCEL_DATE         AS DC_CANCEL_DATE,
         PO.CANCEL_DATE            AS CANCEL_DATE,
         PS.TOTAL_QUANTITY_ORDERED AS TOTAL_QUANTITY_ORDERED,
         PS.TOTAL_DOLLARS_ORDERED  AS TOTAL_DOLLARS_ORDERED,
         PS.WAREHOUSE_LOCATION_ID  AS WAREHOUSE_LOCATION_ID,
         PD.UPC_CODE               AS UPC_CODE,
         PS.BUCKET_ID              AS BUCKET_ID,
         PS.SALES_ORDER_ID         AS STO
    FROM PS
   left outer JOIN PO
      ON PS.PO_ID = PO.PO_ID
     AND PS.CUSTOMER_ID = PO.CUSTOMER_ID
     AND PS.ITERATION = PO.ITERATION
   left outer JOIN PSDET PD
      ON PS.PICKSLIP_ID = PD.PICKSLIP_ID
   left outer join ship ship on ps.shipping_id = ship.shipping_id
   WHERE PS.TRANSFER_DATE IS NULL
     and PD.transfer_date is null 
     and PS.pickslip_cancel_date is null
    AND ps.po_id = :po_id
            And ps.Customer_id = :Customer_id
            AND ps.pickslip_import_date >= :FromDate AND ps.pickslip_import_date < :ToDate + 1
	        AND ship.ship_date >= :ship_date
            AND ship.ship_date < :ship_date + 1
            AND PO.START_DATE >= :from_delivery_date
            AND PO.START_DATE < :to_delivery_date + 1
            AND ps.vwh_id = :vwh_id
            AND PS.REPORTING_STATUS=:reporting_status
            AND 1 = 2
            AND ps.iteration = :iteration
            AND po.dc_cancel_date >= :from_dc_cancel_date
            AND po.dc_cancel_date < :to_dc_cancel_date+1
            AND ps.pickslip_prefix = :pickslip_prefix
            AND ps.label_id = :label_id
            AND :WAREHOUSE_LOCATION_ID
            AND ps.export_flag IS NULL
            AND ps.export_flag = 'Y'
             AND ps.customer_dc_id = :customer_dc_id 
            AND ps.customer_store_id = :customer_store_id
             and not exists (select 1
                            from edi_753_754_ps eps
                             where eps.pickslip_id = ps.pickslip_id
                               and eps.Edi753_Sent_On is not null)
                            and ps.edi_pickslip = 'Y'
              

  UNION ALL
  SELECT DP.PICKSLIP_ID AS PICKSLIP_ID,
         DP.CUSTOMER_ORDER_ID AS PO_ID,
         DP.CUSTOMER_ID AS CUSTOMER_ID,
         DP.CUSTOMER_STORE_ID AS CUSTOMER_STORE_ID,
         DP.CUSTOMER_DIST_CENTER_ID AS CUSTOMER_DC_ID,
         'IN ORDER BUCKET' AS REPORTING_STATUS,
         DP.PRINT_DATE AS PRINT_DATE,
         DP.ONLOT_NUMBER AS ONLOT_NUMBER,
         DP.VWH_ID AS VWH_ID,
         DP.DELIVERY_DATE AS START_DATE,
         DP.DC_CANCEL_DATE AS DC_CANCEL_DATE,
         DP.CANCEL_DATE AS CANCEL_DATE,
         DP.TOTAL_QUANTITY_ORDERED AS TOTAL_QUANTITY_ORDERED,
         DP.TOTAL_DOLLARS_ORDERED AS TOTAL_DOLLARS_ORDERED,
         DP.WAREHOUSE_LOCATION_ID AS WAREHOUSE_LOCATION_ID,
         DPD.UPC_CODE AS UPC_CODE,
         NULL AS BUCKET_ID,
         DP.SALES_ORDER_ID AS STO
    FROM DEM_PICKSLIP DP
   INNER JOIN DEM_PICKSLIP_DETAIL DPD
      ON DP.PICKSLIP_ID = DPD.PICKSLIP_ID
   WHERE 1 = 1
          AND 1=2
     AND dp.delivery_date >= :from_delivery_date
    AND dp.delivery_date < :to_delivery_date + 1
            AND DP.CUSTOMER_ORDER_ID = :po_id
            And DP.Customer_id = :Customer_id
            AND DP.pickslip_import_date >= :FromDate AND DP.pickslip_import_date < :ToDate + 1
            AND DP.vwh_id = :vwh_id
            AND DP.dc_cancel_date >= :from_dc_cancel_date
            AND DP.dc_cancel_date < :to_dc_cancel_date+1
            AND DP.pickslip_prefix = :pickslip_prefix
            AND DP.PICKSLIP_TYPE = :label_id
            AND 1=2
            AND DP.PS_STATUS_ID = '1'
            AND DP.export_flag IS NULL
            AND DP.export_flag = 'Y'  
            AND DP.customer_dist_center_id =:customer_dc_id
            AND DP.Customer_Store_Id =:customer_store_id
            AND :WAREHOUSE_LOCATION_ID 
                 ),
PICKSLIP_DETAIL AS
 (SELECT OPD.PICKSLIP_ID,
         MAX(OPD.STO) AS STO,
         COUNT(UNIQUE OPD.UPC_CODE) AS UPC_CODE,
         MIN(OPD.PO_ID) AS PO_ID,
         MIN(OPD.CUSTOMER_ID) AS CUSTOMER_ID,
         MIN(C.NAME) AS CUSTOMER_NAME,
         MIN(OPD.CUSTOMER_STORE_ID) AS CUSTOMER_STORE_ID,
         MIN(OPD.CUSTOMER_DC_ID) AS CUSTOMER_DC_ID,
         MIN(OPD.REPORTING_STATUS) AS REPORTING_STATUS,
         MIN(OPD.PRINT_DATE) AS PRINT_DATE,
         MIN(OPD.ONLOT_NUMBER) AS ONLOT_NUMBER,
         MIN(OPD.VWH_ID) AS VWH_ID,
         MIN(OPD.START_DATE) AS START_DATE,
         MIN(OPD.DC_CANCEL_DATE) AS DC_CANCEL_DATE,
         MIN(OPD.CANCEL_DATE) AS CANCEL_DATE,
         MIN(OPD.WAREHOUSE_LOCATION_ID) AS WAREHOUSE_LOCATION_ID,
         MAX(OPD.TOTAL_QUANTITY_ORDERED) AS TOTAL_QUANTITY_ORDERED,
         MAX(OPD.TOTAL_DOLLARS_ORDERED) AS TOTAL_DOLLARS_ORDERED,
         RTRIM(SYS.STRAGG(UNIQUE(NVL2(TV.DESCRIPTION,
                                      (TV.DESCRIPTION || ', '),
                                      ''))),
               ', ') AS VAS,
         MAX(OPD.BUCKET_ID) AS BUCKET_ID
    FROM OPEN_PICKSLIP_DATA OPD
    LEFT OUTER JOIN PS_VAS PV
      ON OPD.PICKSLIP_ID = PV.PICKSLIP_ID
    LEFT OUTER JOIN TAB_VAS TV
      ON PV.VAS_ID = TV.VAS_CODE
    LEFT OUTER JOIN CUST C
      ON OPD.CUSTOMER_ID = C.CUSTOMER_ID
  WHERE 1=1
       
                and pv.vas_id is not null        
                 and pv.vas_id =:vas_id    
             
           ANd account_type NOT IN ('RTL')      
     AND account_type NOT IN ('IDC')   
   GROUP BY OPD.PICKSLIP_ID),
BOX_DETAIL AS
 (SELECT MAX(B.PICKSLIP_ID) AS PICKSLIP_ID,
         B.UCC128_ID AS UCC128_ID,
         SUM(BD.CURRENT_PIECES) AS TOTAL_PIECES_IN_BOX,
         SUM(BD.CURRENT_PIECES * BD.EXTENDED_PRICE) AS ESTIMATED_DOLLARS,
         MAX(SC.EMPTY_WT) +
         SUM((MSKU.WEIGHT_PER_DOZEN / 12) * NVL(BD.CURRENT_PIECES,bd.expected_pieces)) AS TOTAL_BOX_WEIGHT,
         MAX(SC.OUTER_CUBE_VOLUME) AS BOX_VOLUME
    FROM BOX B
   INNER JOIN SKUCASE SC
      ON B.CASE_ID = SC.CASE_ID
   INNER JOIN PS
      ON B.PICKSLIP_ID = PS.PICKSLIP_ID
   INNER JOIN BOXDET BD
      ON B.UCC128_ID = BD.UCC128_ID
     AND B.PICKSLIP_ID = BD.PICKSLIP_ID
   INNER JOIN MASTER_SKU MSKU
      ON BD.SKU_ID = MSKU.SKU_ID
   WHERE ps.transfer_date is null 
     AND  B.STOP_PROCESS_DATE IS NULL
     AND BD.STOP_PROCESS_DATE IS NULL
      AND ps.vwh_id = :vwh_id
            AND PS.REPORTING_STATUS=:reporting_status
            AND 1 = 2
            AND ps.iteration = :iteration
              AND ps.po_id = :po_id
            And ps.Customer_id = :Customer_id
            AND ps.pickslip_prefix = :pickslip_prefix
            AND ps.label_id = :label_id
            AND ps.vwh_id = :vwh_id
            AND :WAREHOUSE_LOCATION_ID
            AND ps.export_flag IS NULL
            AND ps.export_flag = 'Y'
             AND ps.customer_dc_id = :customer_dc_id 
            AND ps.customer_store_id = :customer_store_id
   GROUP BY B.UCC128_ID)
SELECT PDET.PICKSLIP_ID as pickslip_id,
       max(PDET.STO) AS STO,
       MAX(PDET.UPC_CODE) AS TOTAL_UPC,
       MAX(PDET.PO_ID) as po_id,
       MAX(PDET.CUSTOMER_ID) as customer_id,
       MAX(PDET.CUSTOMER_NAME) as customer_name,
       MAX(PDET.CUSTOMER_STORE_ID) as customer_store_id,
       MAX(PDET.CUSTOMER_DC_ID) as customer_dc_id,
       MAX(PDET.REPORTING_STATUS) as reporting_status,
       MAX(PDET.PRINT_DATE) as print_date,
       MAX(PDET.ONLOT_NUMBER) as onlot_number,
       MAX(PDET.VWH_ID) as vwh_id,
       MAX(PDET.START_DATE) as start_date,
       MAX(PDET.DC_CANCEL_DATE) as dc_cancel_date,
       MAX(PDET.CANCEL_DATE) as cancel_date,
       MAX(PDET.WAREHOUSE_LOCATION_ID) as warehouse_location_id,
       MAX(PDET.VAS) as vas,
       MAX(PDET.TOTAL_QUANTITY_ORDERED) as total_quantity_ordered,
       MAX(PDET.TOTAL_DOLLARS_ORDERED) as total_dollars_ordered,
       MAX(PDET.BUCKET_ID) as bucket_id,
       COUNT(UNIQUE BDET.UCC128_ID) AS TOTAL_PICKSLIP_BOXES,
       SUM(BDET.TOTAL_PIECES_IN_BOX) AS TOTAL_PIECES_IN_PICKSLIP,
       SUM(BDET.ESTIMATED_DOLLARS) AS TOTAL_ESTIMATED_DOLLARS,
       ROUND(SUM(BDET.TOTAL_BOX_WEIGHT), 4) AS TOTAL_PICKSLIP_WEIGHT,
       ROUND(SUM(BDET.BOX_VOLUME), 4) AS PICKSLIP_VOLUME
  FROM PICKSLIP_DETAIL PDET
  LEFT OUTER JOIN BOX_DETAIL BDET
    ON PDET.PICKSLIP_ID = BDET.PICKSLIP_ID
 GROUP BY PDET.PICKSLIP_ID

WITH Q1 AS
 (SELECT PS.PICKSLIP_ID,
         SUM((MSKU.WEIGHT_PER_DOZEN / 12) * BD.CURRENT_PIECES) AS SKU_WEIGHT,
         SUM((MSKU.VOLUME_PER_DOZEN / 12) * BD.CURRENT_PIECES) AS SKU_VOLUME
    FROM PS
   INNER JOIN BOX B
      ON PS.PICKSLIP_ID = B.PICKSLIP_ID
   INNER JOIN BOXDET BD
      ON B.UCC128_ID = BD.UCC128_ID
     AND B.PICKSLIP_ID = BD.PICKSLIP_ID
   INNER JOIN MASTER_SKU MSKU ON BD.UPC_CODE = MSKU.UPC_CODE
   WHERE PS.PICKSLIP_ID ='16548363'
   GROUP BY PS.PICKSLIP_ID),
WIGHT_BOX AS
 (SELECT B.PICKSLIP_ID,
         SUM(SC.EMPTY_WT) AS EMPTY_WIGHT,
         SUM(SC.OUTER_CUBE_VOLUME)
    FROM BOX B
   INNER JOIN SKUCASE SC
      ON B.CASE_ID = SC.CASE_ID
   WHERE B.STOP_PROCESS_DATE IS NULL
    and b.pickslip_id ='16548363'
    GROUP BY B.PICKSLIP_ID)

SELECT *
  FROM Q1
 LEFT OUTER JOIN WIGHT_BOX BW ON Q1.PICKSLIP_ID = BW.PICKSLIP_ID;

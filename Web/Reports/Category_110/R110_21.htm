R110.21 : Boxes of a customer in different areas

Author : Sanjeev kumar

Mentor : Hemant K. Varshney
$Revision: 7233 $
Status: Reviewed
Tested BY : Sanjeev Kumar

Report helps while shipping orders of customer. It lists all pallets (area wise) to be shipped. Further it helps to locate boxes (non-palletized). You can filter information on the basis of PO, BOL, wave and ATS date etc.Now you get to know count of boxes specific VAS for each pallet with the helps of VAS filter.

Overview

  • This report shows palletized and non-palletized boxes of open orders; customer, building and area wise.
  • As this report is showing area wise palletized and non-palletized boxes so this report will serve a great job for consolidating orders while loading the boxes of an order on to the truck.
  • By using this report users can also find out those pallets which are ready for shipping but not yet reached to door area. Such pallets will be displayed with their location id so that the report user can print this information’s and inform someone to pull those pallets to the door area form the specified locations.
  • This report can be run for a customer with different combinations of the following filters like, Purchase Order, Bill Of Lading, Customer DC, Customer Store and DC Cancel Date range.
  • In order to see customer’s boxes in specific building, building filter can be applied. By default report shows boxes of all building.
  • By default report shows box information Customer wise you can run the report to display same information PO wise.

This report displays the following useful information for the orders being process in the warehouse.:-

  • Area wise palletized and non- palletized boxes are being shown separately in three outputs ‘BOXES NOT ON PALLET’, BOXES ON PALLET and BOXES ON TEMPORARY PALLET
  • BOXES NOT ON PALLET’:- This grid is showing those boxes which are not on the pallet.
  • BOXES ON PALLET: - This grid is showing details of real pallets.
  • BOXES ON TEMPORARY PALLET:- This grid is showing the details of temporary pallets.
  • Report showing pallet wise details (real or temporary pallet) of the boxes, if the pallet is containing boxes of same criteria then report will also show the criteria like Wave, PO, BOL and DC of the boxes on the pallet. In the same grids that is showing palletized boxes if any pallet is having mixed criteria boxes then the report will indicate this by showing MULTIPLE WAVEs (buckets) or MULTIPLE DCs or MULTIPLE BOLs.

Note :-- You need to pass at least one filter from 'boxes of' section.

Algorithm for "Boxes not on pallets"

Get information of non shipped orders of customer from PS table. Retreving name of customer and building from ps table. Retreiving area and  total boxes in any areas from box and boxdet table and load id from edi_753_754_ps table.

Query's documentation:

Boxes Not On Pallet 
   WITH Q1 AS
 (SELECT NVL(CASE
               WHEN B.PITCHING_END_DATE IS NOT NULL AND B.VERIFY_DATE IS NULL AND
                    PV.VAS_ID IS NOT NULL and b.ia_id != :RedBoxArea  THEN
                'VAS'
               ELSE
                B.IA_ID
             END,
             'NON PHYSICAL BOXES') AS BOX_AREA_ID,
         EDIPS.LOAD_ID AS LOAD_ID,
         IA.PROCESS_FLOW_SEQUENCE AS PROCESS_FLOW_SEQUENCE,
         PS.CUSTOMER_ID || ' : ' || C.NAME AS NAME,
         PS.CUSTOMER_ID AS CUSTOMER_ID,
         PS.WAREHOUSE_LOCATION_ID AS BUILDING,
         MS.LABEL_ID AS LABEL_ID,
         B.UCC128_ID AS UCC128_ID
    FROM BOX B
   INNER JOIN BOXDET BD
      ON B.UCC128_ID = BD.UCC128_ID
     AND B.PICKSLIP_ID = BD.PICKSLIP_ID
    LEFT OUTER JOIN MASTER_SKU MSKU
      ON BD.UPC_CODE = MSKU.UPC_CODE
    LEFT OUTER JOIN MASTER_STYLE MS
      ON MSKU.STYLE = MS.STYLE
    LEFT OUTER JOIN IA IA
      ON B.IA_ID = IA.IA_ID
    LEFT OUTER JOIN PS
      ON B.PICKSLIP_ID = PS.PICKSLIP_ID
    LEFT OUTER JOIN CUST C
      ON PS.CUSTOMER_ID = C.CUSTOMER_ID
    LEFT OUTER JOIN EDI_753_754_PS EDIPS
      ON PS.PICKSLIP_ID = EDIPS.PICKSLIP_ID
    LEFT OUTER JOIN PS_VAS PV
      ON PS.PICKSLIP_ID = PV.PICKSLIP_ID
    LEFT OUTER JOIN TAB_VAS TV
      ON PV.VAS_ID = TV.VAS_CODE
    LEFT OUTER JOIN PO
      ON PS.CUSTOMER_ID = PO.CUSTOMER_ID
     AND PS.PO_ID = PO.PO_ID
     AND PS.ITERATION = PO.ITERATION
    LEFT OUTER JOIN SHIP S
      ON PS.SHIPPING_ID = S.SHIPPING_ID
   WHERE PS.TRANSFER_DATE IS NULL
     AND B.PALLET_ID IS NULL
     AND B.STOP_PROCESS_DATE IS NULL
     AND BD.STOP_PROCESS_DATE IS NULL 
     AND PS.customer_id IN (:customer_id0,:customer_id1)
     AND PS.WAREHOUSE_LOCATION_ID IN (:WAREHOUSE_LOCATION_ID0)  
    )
SELECT Q.BOX_AREA_ID as BOX_AREA_ID,
       MAX(Q.LOAD_ID) as LOAD_ID,
       DECODE(Q.BOX_AREA_ID, 'VAS', 25, Q.PROCESS_FLOW_SEQUENCE) AS FLOW_SEQUENCE,
       MAX(Q.NAME) AS NAME,
       Q.CUSTOMER_ID AS CUSTOMER_ID,
       Q.BUILDING AS BUILDING,
       MIN(Q.LABEL_ID) AS MIN_LABEL_ID,
       MAX(Q.LABEL_ID) AS MAX_LABEL_ID,
       COUNT(DISTINCT Q.LABEL_ID) AS COUNT_LABEL_ID,
       COUNT(DISTINCT Q.UCC128_ID) AS TOTAL_BOXES
  FROM Q1 Q
 GROUP BY DECODE(Q.BOX_AREA_ID, 'VAS', 25, Q.PROCESS_FLOW_SEQUENCE),
          Q.BOX_AREA_ID,
          Q.BUILDING,
          Q.CUSTOMER_ID
         ORDER BY MAX(Q.NAME) ASC, FLOW_SEQUENCE ASC NULLS FIRST

Boxes On  Pallet which is started From 'P' and which is not group by on PO 
 
 SELECT B.PALLET_ID PALLET_ID,
       MAX(B.LOCATION_ID) AS LOCATION_ID,
       CASE
         WHEN B.PITCHING_END_DATE IS NOT NULL AND B.VERIFY_DATE IS NULL AND
              PV.VAS_ID IS NOT NULL and b.ia_id != :RedBoxArea
            THEN
          'VAS'
         ELSE
          B.IA_ID
       END AS IA_ID,
       PS.CUSTOMER_ID ||' : '|| C.NAME AS CUSTOMER_NAME,
       RTRIM(SYS.STRAGG(UNIQUE(NVL2(TV.DESCRIPTION, (TV.DESCRIPTION || ', '), ''))),', ') AS VAS,
       CASE WHEN COUNT(UNIQUE PS.SALES_ORDER_ID) >1 THEN
       '*****'
       ELSE
       TO_CHAR(MAX(PS.SALES_ORDER_ID))
       END AS SALES_ORDER_ID,
       PS.WAREHOUSE_LOCATION_ID AS WAREHOUSE_LOCATION_ID,
       MAX(S.SHIP_DATE) AS SHIP_DATE,
       MAX(PO.DC_CANCEL_DATE) AS DC_CANCEL_DATE,
       MAX(edips.LOAD_ID) AS LOAD_ID,
       MIN(MS.LABEL_ID) AS MIN_LABEL_ID,
       MAX(MS.LABEL_ID) AS MAX_LABEL_ID,
       COUNT(DISTINCT MS.LABEL_ID) AS COUNT_LABEL_ID,
       MAX(S.PARENT_SHIPPING_ID) AS MAX_SHIPPING_ID,
       COUNT(DISTINCT S.PARENT_SHIPPING_ID) AS COUNT_SHIPPING_ID,
       MAX(PS.BUCKET_ID) AS MAX_BUCKET,
       COUNT(DISTINCT PS.BUCKET_ID) AS COUNT_OF_BUCKETS,
       MAX(PS.PO_ID) AS MAX_PO,
       COUNT(DISTINCT PS.PO_ID) AS COUNT_OF_POS,
       MAX(PS.CUSTOMER_DC_ID) AS MAX_DC,
       COUNT(DISTINCT PS.CUSTOMER_DC_ID) AS COUNT_OF_DCS,
       MAX(B.MODIFIED_BY) KEEP (DENSE_RANK LAST ORDER BY b.date_modified) AS USER_NAME,
       COUNT(DISTINCT B.UCC128_ID) AS Total_Boxes
  FROM BOX B
 INNER JOIN BOXDET BD
    ON B.UCC128_ID = BD.UCC128_ID
   AND B.PICKSLIP_ID = BD.PICKSLIP_ID
  LEFT OUTER JOIN MASTER_SKU MSKU
    ON BD.UPC_CODE = MSKU.UPC_CODE
  LEFT OUTER JOIN MASTER_STYLE MS
    ON MSKU.STYLE = MS.STYLE
  LEFT OUTER JOIN PS
    ON B.PICKSLIP_ID = PS.PICKSLIP_ID
  LEFT OUTER JOIN PS_VAS PV
    ON PS.PICKSLIP_ID = PV.PICKSLIP_ID
  INNER JOIN tab_vas tv on
            pv.vas_id = tv.vas_CODE
  LEFT OUTER JOIN CUST C ON 
    PS.CUSTOMER_ID = C.CUSTOMER_ID
  LEFT OUTER JOIN SHIP S
    ON PS.SHIPPING_ID = S.SHIPPING_ID
  LEFT OUTER JOIN edi_753_754_ps edips
    ON PS.PICKSLIP_ID = edips.PICKSLIP_ID
  LEFT OUTER JOIN PO
    ON PS.CUSTOMER_ID = PO.CUSTOMER_ID
   AND PS.PO_ID = PO.PO_ID
   AND PS.ITERATION = PO.ITERATION
 WHERE PS.TRANSFER_DATE IS NULL
   AND PALLET_ID LIKE 'P%'
     
    and pv.vas_id is not null        
    and pv.vas_id =:vas_id    
    
   AND B.STOP_PROCESS_DATE IS NULL
   AND BD.STOP_PROCESS_DATE IS NULL
   and s.appointment_id =:appointment_id
   and (:PO_ID)
   AND :SHIPPING_ID
   AND :customer_id
   AND :WAREHOUSE_LOCATION_ID
   AND :customer_store_id
   and :BUCKET_ID
   and (:customer_dc_id) 
   AND edips.ats_date >=CAST(:ats_date AS DATE)
   AND edips.ats_date <=CAST(:ats_date AS DATE) + 1
   AND PO.DC_CANCEL_DATE >= CAST(:FROM_DC_CANCEL_DATE AS DATE) 
   AND PO.DC_CANCEL_DATE < CAST(:TO_DC_CANCEL_DATE AS DATE) + 1
GROUP BY B.PALLET_ID,CASE
         WHEN B.PITCHING_END_DATE IS NOT NULL AND B.VERIFY_DATE IS NULL AND
              PV.VAS_ID IS NOT NULL and b.ia_id != :RedBoxArea
            THEN
          'VAS'
         ELSE
          B.IA_ID
       END, ps.warehouse_location_id, PS.CUSTOMER_ID ||' : '|| C.NAME

        RTRIM(SYS.STRAGG(UNIQUE(NVL2(TV.DESCRIPTION, (TV.DESCRIPTION || ', '), ''))),', ') AS VAS,
       CASE WHEN COUNT(UNIQUE PS.SALES_ORDER_ID) >1 THEN
       '*****'
       ELSE
       TO_CHAR(MAX(PS.SALES_ORDER_ID))
       END AS SALES_ORDER_ID,
       PS.WAREHOUSE_LOCATION_ID AS WAREHOUSE_LOCATION_ID,
       MAX(S.SHIP_DATE) AS SHIP_DATE,
       MAX(PO.DC_CANCEL_DATE) AS DC_CANCEL_DATE,
       MAX(edips.LOAD_ID) AS LOAD_ID,
       MIN(MS.LABEL_ID) AS MIN_LABEL_ID,
       MAX(MS.LABEL_ID) AS MAX_LABEL_ID,
       COUNT(DISTINCT MS.LABEL_ID) AS COUNT_LABEL_ID,
       MAX(S.PARENT_SHIPPING_ID) AS MAX_SHIPPING_ID,
       COUNT(DISTINCT S.PARENT_SHIPPING_ID) AS COUNT_SHIPPING_ID,
       MAX(PS.BUCKET_ID) AS MAX_BUCKET,
       COUNT(DISTINCT PS.BUCKET_ID) AS COUNT_OF_BUCKETS,
       PS.PO_ID AS MAX_PO,
       MAX(PS.CUSTOMER_DC_ID) AS MAX_DC,
       COUNT(DISTINCT PS.CUSTOMER_DC_ID) AS COUNT_OF_DCS,
       MAX(B.MODIFIED_BY) KEEP (DENSE_RANK LAST ORDER BY b.date_modified) AS USER_NAME,
       COUNT(DISTINCT B.UCC128_ID) AS Total_Boxes
  FROM BOX B
 INNER JOIN BOXDET BD
    ON B.UCC128_ID = BD.UCC128_ID
   AND B.PICKSLIP_ID = BD.PICKSLIP_ID
  LEFT OUTER JOIN MASTER_SKU MSKU
    ON BD.UPC_CODE = MSKU.UPC_CODE
  LEFT OUTER JOIN MASTER_STYLE MS
    ON MSKU.STYLE = MS.STYLE
  LEFT OUTER JOIN PS
    ON B.PICKSLIP_ID = PS.PICKSLIP_ID
   LEFT OUTER JOIN PS_VAS PV
    ON PS.PICKSLIP_ID = PV.PICKSLIP_ID
   INNER JOIN tab_vas tv on
            pv.vas_id = tv.vas_CODE
  LEFT OUTER JOIN CUST C ON 
    PS.CUSTOMER_ID = C.CUSTOMER_ID
  LEFT OUTER JOIN SHIP S
    ON PS.SHIPPING_ID = S.SHIPPING_ID
  LEFT OUTER JOIN edi_753_754_ps edips
    ON PS.PICKSLIP_ID = edips.PICKSLIP_ID
  LEFT OUTER JOIN PO
    ON PS.CUSTOMER_ID = PO.CUSTOMER_ID
   AND PS.PO_ID = PO.PO_ID
   AND PS.ITERATION = PO.ITERATION
 WHERE PS.TRANSFER_DATE IS NULL
   AND PALLET_ID NOT LIKE 'P%'
   AND B.STOP_PROCESS_DATE IS NULL
   AND BD.STOP_PROCESS_DATE IS NULL
    
   and pv.vas_id is not null        
   and pv.vas_id =:vas_id    
   
   and s.appointment_id =:appointment_id
   and :PO_ID
   AND :SHIPPING_ID
   AND :customer_id
   AND :WAREHOUSE_LOCATION_ID
   AND :customer_store_id
   and (:BUCKET_ID)
   and (:customer_dc_id) 
       AND edips.ats_date >=CAST(:ats_date AS DATE)
     AND edips.ats_date <=CAST(:ats_date AS DATE) + 1
   AND PO.DC_CANCEL_DATE >= CAST(:FROM_DC_CANCEL_DATE AS DATE) 
AND PO.DC_CANCEL_DATE < CAST(:TO_DC_CANCEL_DATE AS DATE) + 1
 GROUP BY B.PALLET_ID,CASE
         WHEN B.PITCHING_END_DATE IS NOT NULL AND B.VERIFY_DATE IS NULL AND
              PV.VAS_ID IS NOT NULL and b.ia_id != :RedBoxArea
            THEN
          'VAS'
         ELSE
          B.IA_ID
       END,ps.warehouse_location_id,PS.PO_ID, PS.CUSTOMER_ID ||' : '|| C.NAME

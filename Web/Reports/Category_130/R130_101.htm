This report displays the total number of cartons and total number of pieces for each label in a specified area, virtual warehouse and warehouse location.

Overview

This report displays the total number of cartons and pieces for the specified Area, Virtual Warehouse and Warehouse Location.

Algorithm 

For each label retrieved from master_style table, count of number of cartons from src_carton and sum of number of pieces from src_carton_detail table are reterieved on the basis of passed area, virtual warehouse and warehouse location. Table master_storage_location used in the table for retrieving warehouse location in the report.
Item Summary Sort Sequence Detail
Label No 2 Style Label of the SKU
Quality No Quality Code
#Cartons Count 1 Total number of Cartons having SKUs of the corresponding Label
#Pieces Sum   Total number of SKUs in the corresponding Cartons.

Parameters

Name Type Default Description
Area String All Area, where the Carton is currently lying
Virtual Warehouse String All Virtual Warehouse Id for the corresponding Area.
Building String All Building of Area
Quality     Quality Code

Performance Detail

Expected Time for the report output is 1 second

This report will exhibit linear performance with respect to the number of records in src_carton table which qualify for the passed area. There is index based scan on the table src_carton, src_carton_detail, master_style and master_storage_location. Index CTN_STORAGE_I is used in the table src_carton, CTNDET_CTN_FK_I is used in the table src_carton_detail, STLOC_PK is used in the table master_storage_location and STY_PK is used in the table table master_style. We observed that the average query time of 1 second

Query's documentation:

     SELECT COUNT(DISTINCT src_carton.CARTON_ID) AS no_of_carton,
       SUM(SRC_CARTON_DETAIL.QUANTITY) AS no_of_pieces,
       MASTER_STYLE.LABEL_ID AS label_id,
       NVL(ms.Warehouse_Location_Id,'Unknown') AS warehouse_loc,
       src_carton.vwh_id as vwh_id,
       src_carton.quality_code as quality_code
  FROM src_carton src_carton
  left outer join src_carton_detail src_carton_detail on
   src_carton.carton_id = src_carton_detail.carton_id
  LEFT OUTER JOIN MASTER_SKU  MSKU ON
    SRC_CARTON_DETAIL.SKU_ID = MSKU.SKU_ID
  LEFT OUTER JOIN MASTER_STYLE MASTER_STYLE
    ON MSKU.STYLE = MASTER_STYLE.STYLE
  left outer join master_storage_location ms on
   src_carton.carton_storage_area = ms.storage_area
     AND src_carton.location_id = ms.location_id
 WHERE 1 = 1 
 AND src_carton.CARTON_STORAGE_AREA = :carton_storage_area
 AND src_carton.vwh_id = :vwh_id
 AND ms.Warehouse_Location_Id = :warehouse_location_id
 and ms.Warehouse_Location_Id is null
 AND src_carton.quality_code = :quality_code
 GROUP BY MASTER_STYLE.LABEL_ID, ms.Warehouse_Location_Id, src_carton.vwh_id, src_carton.quality_code

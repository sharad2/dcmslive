The report displays carton details coming from sewing plant for a specific date or date range.

Overview

The report displays inf4ormation about cartons and carton quantity coming from sewing plant for a specify date or date range. This report gives all information about cartons which are coming from sewing plant or already recieved from sewing plant. This report will displays number of cartons and quantities of SKUs with it's sewing plant, shipment id and process date. This report also displays the carton information along with SKU details and pieces stored in the cartons against a particular sewing plant code. by default this report is sorted on Sewing Plant Code and Shipment Id in ascending order.

Algorithm

Get all information of manifest from the src_carton_intransit table.

Parameters

Name Default Detail

To Date Current Date. The date to which you want to display informaion about manifest.

Query's documentation:

Master_table

  • In this query, we are using src_carton_intransit tables to display sewing_plant_code,number_sewing_plant_code,shipment_id ,insert_date,no_of_cartons and quantity. 
    SELECT /*+ index (src CTNINTRAN_INSERT_DATE_I)*/src.sewing_plant_code AS sewing_plant_code,
       src.shipment_id AS shipment_id,
       trunc(src.insert_date) AS insert_date,
       count(distinct(src.carton_id)) AS no_of_cartons,
       sum(src.quantity) AS quantity
  FROM src_carton_intransit src
 WHERE src.insert_date >= :from_date
   AND src.insert_date < :to_date + 1
 GROUP BY src.sewing_plant_code, src.shipment_id, trunc(src.insert_date)

Child_table

  • In this query, we are using src_carton_intransit tables to display sewing_plant_code,number_sewing_plant_code,shipment_id ,no_of_cartons,pieces and SKUs. 
    SELECT /*+ index (src CTNINTRAN_INSERT_DATE_I)*/ src.sewing_plant_code AS sewing_plant_code,
       src.shipment_id AS shipment_id,
       COUNT(distinct(src.carton_id)) AS no_of_cartons,
       SUM(src.quantity) AS pieces,
       src.style AS style,
       src.color AS color,
       src.dimension AS dimension,
       src.sku_size AS sku_size,
       src.source_order_prefix || src.source_order_id AS source_order 
  FROM src_carton_intransit src
 WHERE src.insert_date >= :FROM_DATE
   AND src.insert_date < :TO_DATE + 1
 GROUP BY src.sewing_plant_code ,src.shipment_id ,src.style ,src.color , src.dimension ,src.sku_size , src.source_order_prefix || src.source_order_id

10.07 : DCMS to Richter Export Progress Report

Author : Sanjeev Kumar

$Revision: 5595 $
Reviewed By: Manmohan Bisht
Status: Reviewed
Tested By : Hemant K. Singh

The report displays the number of records that are being exported from DCMS to Richter .

Overview

The report displays the number of records that are waiting to be exported from DCMS to Richter. Through this report one can view current progress of records being exported from DCMS.

Algorithm

Get count of all records for each record name with total number of records from table LOAD_ISI_WM_CHRONO and table LOAD_ISI_WM_CHRONO_POC.

Output

Item Summary Default Sort Sequence Detail
Record Type No 1  
Record Name No 2  The Record Name for each type of records for both tables LOAD_ISI_WM_CHRONO and LOAD_ISI_WM_CHRONO_POC.
No. of Records Sum   The total number of each type of records for both tables LOAD_ISI_WM_CHRONO and LOAD_ISI_WM_CHRONO_POC.

Performance Detail

Expected Time for the report output is 0.5 seconds

There is full scan on the tables LOAD_ISI_WM_CHRONO and LOAD_ISI_WM_CHRONO_POC.

Issues

No Known Issue.

Prerequisites

Private synonyms for the following tables are requried

load_isi_wm_chrono,
load_isi_wm_chrono_poc

Use the below script to create the synonyms

create or replace synonym load_isi_wm_chrono for dcms4.load_isi_wm_chrono;
create or replace synonym load_isi_wm_chrono_poc for dcms4.load_isi_wm_chrono_poc;  

Parameter Technical Details

This Report has no Parameters.

Query's documentation:

  • The query will get the total reords for each type reords from LOAD_ISI_WM_CHRONO. 
                  SELECT isi_wm_chro_record_name "Record Name",
              COUNT(isi_wm_chro_record_name) "No.of Records"
              FROM dcms4.load_isi_wm_chrono
              GROUP BY isi_wm_chro_record_name;
  • The query will get the total reords for each type reords from LOAD_ISI_WM_CHRONO_POC. 
                  SELECT isi_wm_chro_record_name "Record Name POC",
              COUNT(isi_wm_chro_record_name) "No.of Records POC"
              FROM dcms4.load_isi_wm_chrono_poc
              GROUP BY isi_wm_chro_record_name;

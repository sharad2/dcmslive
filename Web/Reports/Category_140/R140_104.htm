For a particular pickslip id, virtual warehouse, label and customer id this report shows the ordered SKU and their current pieces

Overview

This is the drilldown on pickslip id to show the pickslip details. For a particular pickslip id, virtual warehouse, label and customer id the report shows the SKU details along with the carton id (UCC128 id) and pieces currently contained in boxes of the passed pickslip.

Algorithm

Get all the pickslip details from tables master_sku ,box, boxdet and ps.

Output

Item Summary Sort Sequence Detail
Carton Id No 1 UCC128 Id of the box
Style No 2 Style of the SKU contain by the box
Color No   Color of the SKU contain by the box
Dimension No   Dimension of the SKU contain by the box
Size No   Size of the SKU contain by the box
Current Pieces Sum   Piecess currently lying in the box

Parameters

Name Type Default Detail
Pickslip Number None Pickslip Id, whose boxes you wish to see.
Virtual Warehouse String All Virtual Warehouse of the pickslip
Label String All Lable of the SKU
Customer Id String None Customer Id, whose boxes you wish to see.
Warehouse location id String All The particular warehouse location id for which you would like to run the report

Performance Detail

Expected Time for the report output is 1 second

The query will exhibit linear performance with respect to the number of Skus in boxes of Pickslip. The query also does a index based search on master_sku ,box, boxdet and ps. The query took 1 second to display the records.

Query's documentation:

        select b.ucc128_id, 
               ms.style as style,
               ms.color as color,
               ms.dimension as dimension,
               ms.sku_size as sku_size,
               sum(bd.current_pieces) as current_pieces
          from boxdet bd
         inner join box b on bd.pickslip_id = b.pickslip_id
                         and bd.ucc128_id = b.ucc128_id
         inner join ps ps on ps.pickslip_id = b.pickslip_id
         LEFT OUTER JOIN po po
            ON ps.customer_id = po.customer_id
           AND ps.po_id = po.po_id
           AND ps.iteration = po.iteration
         inner join master_sku ms on bd.upc_code = ms.upc_code
         where ps.pickslip_id = cast(:pickslip_id as number)
         AND ps.label_id=cast(:label_id as varchar2(255))
         AND ps.vwh_id = CAST(:vwh_id as varchar2(255)) 
         AND ps.customer_id=CAST(:customer_id as varchar2(255)) 
         and ps.warehouse_location_id =:WhLoc  
         AND po.start_date >= CAST(:delivery_start_date AS DATE)      
         AND po.start_date <= CAST(:delivery_end_date AS DATE)       
         AND po.dc_cancel_date >=CAST(:dc_cancel_start_date AS DATE)
         AND po.dc_cancel_date <=CAST(:dc_cancel_end_date AS DATE) 
         group by b.ucc128_id, 
                  ms.style,
                  ms.color,
                  ms.dimension,
                  ms.sku_size

This report display list of cartons with the SKU detail for specific storage area. This report also display FPK location of the SKU.  User can futher filter information on the basis of label, style, modified date, virtual warehouse, quality, Price season code and building. It report able to see information sift wise.  

Overview

The main goal of this report is to display the comprehensive details of each carton on a pallet or location for the various storage area like AWL,AWO,AWS,REW,RCV,RST,CFD. For a specified storage area,the report displays the Cartons with there SKU details of each carton lying on the pallet or on the location. It also provide the facility to display the cartons for a specified label and if you would like to see pulls generated for the pulls generated areas in a given time period just select pulls generated area and choose the option pulls generated on specified date. This is date when the cartons was modified for any request or changes has been done in the carton. Following are some of the uses of this report

  • At any given point of time exact position of all the cartons awaiting a process can be found out.
  • Bottleneck process can be identified.
  • Efficiency of each department can be found out.
  • You can also see the pulls generated for the pulls generated areas in a given time period.

Algorithm

For a specified storage area Pallet, Location id ,carton id  are retrieved from the src_carton table. Retrieving style, color, dimension, sku_size and req_process_date from src_carton_detail. Retrieve the assigned location of the SKU from IALOC and building of the carton from master_storage_location and tab_inventory_area table.

Technical Documentation

 

Parameter Technical Parameter Technical Details

  • Carton Storage Area: (Type = ID): This is a drop down list parameter control. By default, inventory of all Warehouse id is shown for each area. If your focus is on the inventory in a particular Warehouse, Specify that Virtual warehouse. default value text is "All".
    • Query to fill list values = " select distinct(inventory_storage_area) from tab_inventory_area where tab_inventory_area.stores_what='CTN' order by inventory_storage_area " li>
  • Label (Type = ID): Label assigned to the carton
  • Style(Type =ID):  Style assigned to the carton
  • Summarize Modified Date (Type = Date): Pulls generated for RST area in a given time period This is a date range control.
  • Shift (Type = ID): This is a drop down list parameter control. You can select shift to see pulls generated for RST area in a specific shift. Default value "All".
    • Value = 1, Text = First
    • Value = S, Text = Second
  • Virtual Warehouse (Type = ID): This is a drop down list control. Virtual Warehouse default value is "All".
    • Query to fill list values = "SELECT vwh_id AS vwh_id, vwh_id || ': ' || description AS description FROM tab_virtual_warehouse ORDER BY vwh_id"
  • Quality (Type = ID): This is a drop down list control. Quality Code default value is "All".
    • Query to fill list values = "select tqc.quality_code as qualitycode , tqc.description as qualitydescription from tab_quality_code tqc order by tqc.quality_rank asc"
  • Price Season Code(Type = ID): This is a drop down list control. Price Season Code default value is "All".
    • Query to fill list values = "SELECT T.PRICE_SEASON_CODE AS PRICE_SEASON_CODE, T.DESCRIPTION AS DESCIPTION FROM TAB_PRICE_SEASON T order by PRICE_SEASON_CODE"
  • Building (Type = ID): This is a drop down list control. building default value is "All".
    • Query to fill list values = "WITH Q1 AS (SELECT TWL.WAREHOUSE_LOCATION_ID, TWL.DESCRIPTION FROM TAB_WAREHOUSE_LOCATION TWL UNION SELECT 'Unknown' AS WAREHOUSE_LOCATION_ID, 'Unknown' AS DESCRIPTION FROM TAB_WAREHOUSE_LOCATION TWL) SELECT Q1.WAREHOUSE_LOCATION_ID, (Q1.WAREHOUSE_LOCATION_ID || ':' || Q1.DESCRIPTION) AS DESCRIPTION FROM Q1 ORDER BY 1"

Query's documentation:

  • This Query displays the comprehensive details of each carton for a specified storage area for the date on which cartons were modified. 
                  
WITH CARTON_DETAIL AS
 (SELECT SC.CARTON_ID,
         MS.UPC_CODE,
         SC.LOCATION_ID AS CARTON_LOCATION,
         SC.PALLET_ID,
         SC.MODIFIED_DATE,
         NVL2(SC.SUSPENSE_DATE, 'TRUE', '') AS SUSPENSE_DATE,
         COALESCE(TIA.WAREHOUSE_LOCATION_ID,
                  MSL.WAREHOUSE_LOCATION_ID,
                  'Unknown') AS WAREHOUSE_LOCATION_ID,
         SC.CARTON_STORAGE_AREA,
         tia.short_name,
         SC.VWH_ID,
         MS.STYLE,
         MS.COLOR,
         MS.DIMENSION,
         MS.SKU_SIZE,
         SCD.QUANTITY,
         SCD.req_assign_date,
         sc.quality_code,
         SC.PRICE_SEASON_CODE
    FROM SRC_CARTON SC
   INNER JOIN SRC_CARTON_DETAIL SCD
      ON SC.CARTON_ID = SCD.CARTON_ID
    LEFT OUTER JOIN MASTER_SKU MS
      ON SCD.SKU_ID = MS.SKU_ID
    LEFT OUTER JOIN MASTER_STORAGE_LOCATION MSL
      ON SC.CARTON_STORAGE_AREA = MSL.STORAGE_AREA
     AND SC.LOCATION_ID = MSL.LOCATION_ID
    LEFT OUTER JOIN TAB_INVENTORY_AREA TIA
      ON SC.CARTON_STORAGE_AREA = TIA.INVENTORY_STORAGE_AREA
    WHERE 1=1
     AND sc.carton_storage_area = :CARTON_STORAGE_AREA
     AND ms.style = :style
     AND sc.VWH_ID = :vwh_id
     and sc.quality_code= :quality_code
     and sc.price_season_code =:price_season_code
     AND sc.modified_date >= (:modified_date_from)
     AND sc.modified_date < (:modified_date_to)+1
     AND  :WAREHOUSE_LOCATION_ID
     
),
IALOC_DETAIL AS
 (SELECT I.ASSIGNED_UPC_CODE,
         I.VWH_ID,
         SYS.STRAGG(UNIQUE(I.LOCATION_ID || ', ')) AS assigned_fpk_location
    FROM IALOC I
   WHERE I.ASSIGNED_UPC_CODE IS NOT NULL
   AND i.VWH_ID = :vwh_id
   AND :WAREHOUSE_LOCATION_ID
   GROUP BY I.ASSIGNED_UPC_CODE, I.VWH_ID)
SELECT CD.CARTON_ID,
       CD.UPC_CODE,
       CD.CARTON_LOCATION,
       CD.PALLET_ID,
       CD.MODIFIED_DATE,
       CD.SUSPENSE_DATE,
       CD.WAREHOUSE_LOCATION_ID,
       CD.CARTON_STORAGE_AREA,
       CD.VWH_ID,
       CD.STYLE,
       CD.COLOR,
       CD.DIMENSION,
       CD.SKU_SIZE,
       CD.QUANTITY,
       MST.LABEL_ID,
       cd.short_name,
       ICD.assigned_fpk_location,
       CD.req_assign_date,
       cd.quality_code,
       CD.PRICE_SEASON_CODE
  FROM CARTON_DETAIL CD
  LEFT OUTER JOIN IALOC_DETAIL ICD
    ON ICD.ASSIGNED_UPC_CODE = CD.UPC_CODE
   AND ICD.VWH_ID = CD.VWH_ID
  LEFT OUTER JOIN MASTER_STYLE MST ON 
            CD.STYLE = MST.STYLE
where 1=1
AND mst.Label_ID = :LABEL_ID

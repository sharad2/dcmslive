This Report provides the information of all the pallets lying in storage area which you will select from the drop down list. The report provides insight about the pallets and the cartons awaiting some process.

Overview

Report provides the information of all the pallets lying in storage area which you will select from the drop down list. For each Pallet lying in the selected carton storage area the report shows the Pallet ID, Pallet Created Date(Modified Date) i.e. the date when the pallet was last modified /created and number of cartons lying on it.

Algorithm

The report query get all the pallet related information from src_carton table.

Output

Parameters

Performance Detail

Expected Time for the report output is 1 Sec.

The report query is performing linear performance w.r.t. table src_carton. There is only one table used in report src_carton and there is full scan on the src_carton table.

Parameter Technical Details

  • Specify Area: (Type = ID): This is a drop down list parameter control. By default, inventory of all Warehouse id is shown for each area. If your focus is on the inventory in a particular Warehouse, Specify that Warehouse Id. default value text is "All".
    • Query to fill list values = " select distinct(inventory_storage_area) from tab_inventory_area where tab_inventory_area.stores_what='CTN' order by inventory_storage_area "

Parameter Validation

No special validations.

Query's documentation:

  • The query will get all the pallet related information from src_carton table. 
                   SELECT ctn.pallet_id AS pallet_id,
        ctn.quality_code AS quality_code,
        MAX(ctn.location_id) AS location_id,
        COUNT(ctn.carton_id) AS cartons_count,
        MIN(ctn.modified_date) AS modified_date,
        CTN.CARTON_STORAGE_AREA,
        MAX(TIA.SHORT_NAME) AS AREAS,
        COALESCE(TIA.WAREHOUSE_LOCATION_ID,
                MSL.WAREHOUSE_LOCATION_ID,
                'Unknown') AS WAREHOUSE_LOCATION_ID
        FROM src_carton ctn
        LEFT OUTER JOIN MASTER_STORAGE_LOCATION MSL
        ON CTN.CARTON_STORAGE_AREA = MSL.STORAGE_AREA
        AND CTN.LOCATION_ID = MSL.LOCATION_ID
        INNER JOIN TAB_INVENTORY_AREA TIA
        ON CTN.CARTON_STORAGE_AREA = TIA.INVENTORY_STORAGE_AREA
        WHERE ctn.pallet_id IS NOT NULL
        AND ctn.carton_storage_area = :carton_storage_area
        GROUP BY ctn.pallet_id, ctn.quality_code,
          CTN.CARTON_STORAGE_AREA,
          COALESCE(TIA.WAREHOUSE_LOCATION_ID,
                   MSL.WAREHOUSE_LOCATION_ID,
                   'Unknown');

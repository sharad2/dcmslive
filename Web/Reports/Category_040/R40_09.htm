40.09 : Cartons on a Pallet

Author : Sanjeev Kumar

$Revision: 840 $
Reviewed By : Ritesh Verma

For user specified pallet id, report displays the Carton ID, its pull priority (priority at which the request to be processed), SKU level information, whether they are assigned on any FPK location or not for each carton lying on the pallet.

Overview

Important: The report is helpful in displaying the information about the cartons on a specified pallet so that DC personnel know which carton is on which pallet and if the carton goes missing it could be tracked back as well. .

Validation details

Query: 
 
        select
       b.carton_id           as carton_id,
       b.ucc128_id           as box_id,
       b.pallet_id           as pallet_id,
       b.pickslip_id         as pickslip_id,
       b.ia_id               as area 
       from box b 
       where b.pallet_id=:pallet_id
Query: 
 
          select sc.pallet_id as pallet_id,
              sc.carton_id as caron_id,
              sc.last_pulled_date as last_pulled_date,
              SC.CARTON_STORAGE_AREA AS AREA
           from src_carton sc
       where sc.pallet_id=:pallet_id
Description:
The above query display the Pallet Detail specified for the carton lying on the pallet.
